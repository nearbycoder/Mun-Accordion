Mun-Accordion

Pure Angular Accordion Menu

Below is examples of my work for the code challenage. If I could make any improvements on this work it would be around the animations of the accordions. In addition the css was compiled scss that I built as a custom grid framework. If you would like to see the full project you can view it on github here
Example API 
<mun-accordion>
  <accordion-item item-title="Heading 1" initially-open="true">
      test
  </accordion-item>
  <accordion-item item-title="Heading 2">
      test
  </accordion-item>
  <accordion-item item-title="Heading 3">
      test
  </accordion-item>
</mun-accordion>
The code above reflects the first accordion below. item-title="Heading 1" and initially-open="true" are attributes that affect both the title of the accordion as well as which accordion is intially opened. Each mun-accordion directive will act on its own and will not affect other instances of the directive.
